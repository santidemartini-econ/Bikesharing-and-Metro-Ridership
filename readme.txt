Raw data should be download from the sources detailed in each folder. Once the folder is download, by running the corresponding codes in order the clean databases will be created. All of the data is public except the one of Mercado Libre, so you will not be able to access to that data and therefore also unable to replicate the results that use that database. 

- 'Argentina' contains the data, codes and output figures and tables from the analysis done in the City of Buenos Aires

- 'CDMX' contains the data, codes and output figures and tables from the analysis done in Mexico City
