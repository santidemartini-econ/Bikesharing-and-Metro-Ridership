- 'final_merge' merges all the databases to produce the final database, both at the daily and at the monthly frequency

- 'meli_proc' processes the data from Mercado Libre

- 'd_bike_subw_cl' creates a database of the distance of each bike station to each subway station

- 'distance_trips' creates a database of the distance of each trip done (in meters)

- 'api_query' retrieves the query results of the availability of bikes in each of the stations in CABA, from the API of the government of Buenos Aires
