- 'Availability Bikes' includes information of the availability of bikes in each station

- 'Distances' includes the databases on the distances between each bikeshare station and each subway station

- 'EOD' contains the results of the travel survey done in 2017

- 'Final' has the final database used in the analysis

- 'Geo' includes the geographic location of the metro stations and the bikeshare stations

- 'Regs' includes the results of the regressions estimated

- 'Ridership' contains the number of trips done every day since 2010 both in the BRT and in the subway, by tap station. 

- 'Trips' includes all the trips done in the bikeshare system since 2010.
